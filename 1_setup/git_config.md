## git config 
### List 
```
> git config --global -l 
> git config --system -l 
> git config --local -l
```

### Edit 
```
> git config --global user.name "Hiroshi Nomura" 
> git config --global user.email "hnomura@foo.boo.com" 
> git config --global color.ui auto 
> git config --global core.editor notepad
```
